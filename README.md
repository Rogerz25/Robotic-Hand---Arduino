# Robotic-Hand---Arduino
Voice Controlled robotic hand

Library Dependencies for Arduino.IDE:
===
1) Adafruit pwm servo driver library
2) DFRobot_DF2301Q

PCA9685 Board Layout:
======================

Arduino - PCA Board
A4 → SDA
A5 → SCL
5V → VCC
GND → GND

Voice Module Layout:
Voice Module pin------Arduino pin
VCC	                  5V
GND                 	GND
SDA / D/T	            A4
SCL / C/R	            A5
